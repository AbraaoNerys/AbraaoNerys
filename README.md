### 📊 SGE — Sistema de Gerenciamento Empresarial
> **Back-end focus** · Java / Node.js · PostgreSQL · REST API

Sistema criado para **microempreendedores individuais (MEIs)** com o objetivo de simplificar a gestão do negócio. O SGE fornece uma visão estratégica da empresa, permitindo decisões baseadas em dados reais.

**Funcionalidades do sistema:**
- 📦 Controle de estoque em tempo real
- 💰 Análise de lucro e prejuízo
- 📈 Dashboard de desempenho de vendedores
- 🏆 Identificação de produtos mais e menos vendidos
- 📉 Relatórios de análise de vendas

> *Projeto pensado para resolver dores reais de pequenos negócios — simplicidade, usabilidade e eficiência como princípios.*
